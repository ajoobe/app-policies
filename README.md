# Ajoobe App Policies

This repository hosts public policy and support pages for Ajoobe applications.

## Contents

- [ActionCam Live Controller — Privacy Policy](actioncam-live-controller/privacy/index.html)
- [ActionCam Live Controller — Support](actioncam-live-controller/support/index.html)

## Purpose

These pages are intended for:
- Google Play Store listing requirements (privacy policy URL)
- User support and troubleshooting
- Developer contact information

## Important

- **No app source code** is stored in this repository.
- These are static HTML pages — no JavaScript, no analytics, no tracking.
- All content is public by design to meet app store policy requirements.

## Deployment

This repository is published via GitHub Pages. Any push to the default branch automatically updates the live site.
